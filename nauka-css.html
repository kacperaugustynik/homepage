Podstawy css

  • class - pozwala na gropowanie elementow odwolujemy sie przez "."
  • id - uzywamy tylko do jednego elementu odwokujemy sie przez "#"
  • "!important" nie stosujemy, nadpisuje wszytskie inne rzeczy i ignoruje i zabija wszytskie inne reguły powoduje chaos
  • Specyficzność css(reguły) - 1.!important 2. style="" 3. #id 4. .class 5. elementy
  • Dodawac klasy do wszytskich elementow i unikać zagniezdzen
  • Nazywnictwo klass:
    • pojedyncze slowa z angielskiego
    • uzwyac kebab charset
    • nazywac zgodnie z ich funkcjonalnoscia na stronie
    • starac sie aby element mial jedna klase(nie zwiekszać bez potrzeby)
    • nie zaczynac od cyferek
  • Jednostki: px - pikesle, % - zalezne od rodzica, vh & vw - procent ale od rozmiaru calego okna a nie z rodzica
